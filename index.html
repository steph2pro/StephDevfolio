Mes Travaux

Application Millearnia

Millearnia est une application d'orientation scolaire conçue pour aider les étudiants à choisir leur parcours académique et professionnel en fonction de leurs compétences et intérêts. Elle propose une interface intuitive et moderne avec une expérience utilisateur optimisée grâce à des outils performants.

Technologies utilisées :

  • React - pour la construction de l'interface utilisateur
  • Clean Architecture - pour une organisation modulaire et scalable
  • React Query - pour la gestion optimisée des requêtes et du cache
  • Yup - pour la validation des formulaires
  • Transactable - pour la gestion fluide des transactions de données
  • Tailwind CSS - pour un design moderne et réactif
  • ShadCN - pour des composants UI élégants et accessibles

Backend de l'application Millearnia

Le backend de Millearnia est conçu pour fournir une API robuste et sécurisée permettant de gérer les données des utilisateurs, les recommandations scolaires et les interactions avec l'application. Il repose sur des technologies modernes garantissant performance, scalabilité et sécurité.

Technologies utilisées :

  • AdonisJS - framework backend Node.js avec une architecture MVC
  • Prisma - ORM moderne pour une gestion efficace de la base de données
  • Vine - pour la validation des données et la sécurité des entrées utilisateur
  • PostgreSQL (via Supabase) - base de données relationnelle performante et évolutive
  • Firebase Authentication - pour une gestion sécurisée des utilisateurs
  • Communication API - API RESTful optimisée pour une interaction fluide avec le frontend

Application ESMOD

L’application ESMOD est un système de gestion complet pour un institut universitaire. Elle permet d’administrer les étudiants, les enseignants, les cours, les inscriptions ainsi que la gestion des notes et des emplois du temps. Développée avec Laravel, elle assure une architecture robuste et sécurisée.

Technologies utilisées :

  • Laravel - framework PHP moderne pour un développement rapide et sécurisé
  • Blade - moteur de templates pour des vues dynamiques et modulaires
  • MySQL - base de données relationnelle pour gérer les utilisateurs et les informations académiques
  • Eloquent ORM - gestion efficace des modèles et des relations entre les données
  • Livewire - pour des interfaces dynamiques sans rechargement de page
  • Bootstrap & Tailwind CSS - pour un design moderne et responsive
  • Gestion des rôles - système d’authentification et d’autorisation pour les étudiants, enseignants et administrateurs
